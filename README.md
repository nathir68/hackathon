# HIRE - AI-Powered Job Portal

HIRE is an advanced, full-stack recruitment platform designed to connect job seekers with opportunities while equipping recruiters and administrators with powerful AI-driven tools to manage the hiring pipeline efficiently.

## 🚀 Key Features

### For Job Seekers
- **Smart Job Search:** Browse and find jobs specifically tailored to your skills and preferences.
- **AI Resume Builder & Scorer:** Build standout resumes and get them evaluated dynamically against industry standards using AI.
- **AI Mock Interviews & Chat:** Practice your interview skills with an interactive AI agent providing real-time technical questions and feedback.
- **Career Predictor:** Discover the best career trajectories based on your current skill set.
- **Saved Jobs & Alerts:** Bookmark your favorite positions and get notified of updates.

### For Recruiters
- **Post & Manage Jobs:** Seamlessly publish job descriptions generated by AI templates to the platform.
- **AI ATS Checker & Candidate Shortlisting:** Automatically filter, score, and rank candidates using intelligent AI models to find the perfect fit faster.
- **Analytics & Pipeline:** Evaluate candidate performance and manage the recruitment lifecycle.

### For Administrators
- **Comprehensive Dashboard:** Monitor total users, jobs, revenues, and application flows across the platform.
- **Manage Roles:** Full control to moderate Admin, Job Seeker, and Recruiter accounts.
- **System Activity:** Stay informed on system events, logs, templates, and overarching AI settings.

## 🛠️ Technology Stack

- **Frontend:** React.js, React Router DOM, Axios, standard dynamic CSS
- **Backend:** Node.js, Express.js
- **Database:** MongoDB Atlas (Mongoose ORM)
- **AI Integration:** Google Gemini AI API (`@google/generative-ai`)
- **Authentication:** JSON Web Tokens (JWT), bcryptjs

## ⚙️ Local Development Setup

### Prerequisities
- Node.js installed locally
- MongoDB Atlas Cluster or Local MongoDB
- Google Gemini API Key

### 1. Backend Setup
Navigate to the backend directory:
```bash
cd backend
```
Install dependencies:
```bash
npm install
```
Set up your environment variables by creating a `.env` file inside `backend/`:
```env
PORT=5000
MONGODB_URI=your_mongodb_connection_string
JWT_SECRET=your_jwt_secret_key
GEMINI_API_KEY=your_gemini_api_key
```
Start the backend server:
```bash
npm start
# (Alternatively, use npm run dev for nodemon)
```

### 2. Frontend Setup
Navigate to the frontend React directory:
```bash
cd HIRE/frontend
```
Install dependencies:
```bash
npm install
```
Start the local development server:
```bash
npm start
```
*The web app should now be accessible locally on `http://localhost:3000`.*

## 📚 Repository Architecture overview
- `/HIRE/frontend/` - Contains all React UI components (`src/Pages`, `src/components`), styles, and state logic.
- `/backend/` - The Express server consisting of API logic (`controllers/`, `routes/`, `models/`), utility scripts, and MongoDB connections.
